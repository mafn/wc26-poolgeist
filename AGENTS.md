# AGENTS.md

Guidance for agents working in this repository:

- Keep the project neutral: do not add private predictions, hard-coded favorites, company-specific data, or biased default picks.
- Use only synthetic or neutral demo data in committed files.
- This is not an official FIFA, World Cup, or Kicktipp project; preserve the no-affiliation disclaimers in user-facing docs.
- Prefer `uv` for environment and command execution when available.
- Run `make format`, `make lint`, and `make test` before committing code changes.
